# Garbo Eventos

Site estático para locação de produtos para festas e eventos.

## Como abrir
Abra `index.html` no navegador.

## Estrutura
- `index.html`: página principal
- `style.css`: estilos
- `script.js`: busca, filtros e galeria
- `produtos.js`: dados do catálogo
- `assets/produtos`: imagens dos produtos
